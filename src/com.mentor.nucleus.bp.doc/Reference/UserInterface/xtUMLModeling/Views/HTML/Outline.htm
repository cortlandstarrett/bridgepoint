Outline View

The Outline view assists the user in understanding the contents of the currently selected diagram.  The information is shown hierarchically and is similar to that of the Model Explorer view.  Whereas the latter shows all of the model elements in the hierarchy, the former shows only those contained with the currently selected diagram.  For example, when looking at the Class Diagram for the MicrowaveOven subsystem, only the classes contained in that subsystem, along with everything below (in the hierarchy) is shown.
